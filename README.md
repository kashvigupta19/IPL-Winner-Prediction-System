# 🏏 IPL Winner Prediction System using Machine Learning

An interactive **AI/ML-based IPL Winner Prediction System** built using **Python, Scikit-learn, and Streamlit**. The project predicts the winning team of an IPL match by analyzing team strength, match conditions, toss details, weather, and other engineered features.

---

## 📌 Project Overview

This project demonstrates a complete end-to-end Machine Learning workflow:

- Data Cleaning & Preprocessing
- Synthetic Match Dataset Generation
- Feature Engineering
- Training Multiple Machine Learning Models
- Model Evaluation & Comparison
- Interactive Streamlit Web Application

The final application allows users to select match conditions and predict the winning team with an easy-to-use interface.

## 🚀 Installation

Clone the repository

```bash
https://github.com/kashvigupta19/IPL-Winner-Prediction-System.git
```

Move into the project directory

```bash
cd IPL-Winner-Prediction-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

## 📊 Dataset

The project uses a player dataset as the base dataset.

Since complete IPL match-level data was unavailable, a synthetic match dataset was generated using:

- Team Strength
- Recent Team Form
- Toss Winner
- Toss Decision
- Weather
- Home Advantage
